# Animeter Web App
[Frontend](https://animeter.vercel.app/) - [Backend](https://animeter.herokuapp.com/)

This repository contains the source code for the Animeter Web Application.

## Setup

```
git clone https://github.com/habbit-tech/habbit-webapp
cd habbit-webapp
yarn
```

## Development Server

```
yarn dev
```
